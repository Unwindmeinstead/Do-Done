# Do.Done - Cursor Mobile Agent Configuration

This project is configured for optimal use with Cursor mobile agents.

## Project Structure
- **Framework**: React 18 with Tailwind CSS
- **Language**: JavaScript (ES6+)
- **Package Manager**: npm
- **Port**: 3000

## Available Branches
- `main` - Production/stable code
- `ui-testing` - Experimental UI changes

## Commands
- `npm start` - Start development server
- `npm run build` - Build for production
- `npm test` - Run tests

## Features Enabled
- ✅ Git integration with branch support
- ✅ Terminal operations
- ✅ File explorer
- ✅ Search functionality
- ✅ React development tools

## Mobile Agent Tips
- Use `git checkout <branch-name>` to switch branches
- Branches should now appear in the mobile agent interface
- All git operations are supported
- File operations respect the .cursorignore patterns
